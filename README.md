# Geometric_lib 

Библиотека для вычисления геометрических параметров фигур.

## Общее описание решения

Проект представляет собой математическую библиотеку для вычисления площадей и периметров различных геометрических фигур. 

## Описание функций

### Файл: circle.py

#### area(r)
 Вычисляет площадь круга.

 Параметры:
 - r (float): Радиус круга.

Возвращает:
 - float: Площадь круга.

Пример вызова:
 result = area(3.5)
 print(result)  # Вывод: 38.48451000647496

#### perimeter(r)
 Вычисляет длину окружности (периметр круга).

 Параметры:
 - r (float): Радиус круга.

 Возвращает:
- float: Длина окружности.

 Пример вызова:
 result = perimeter(3.5)
 print(result)  # Вывод: 21.991148575128552

### Файл: square.py

#### area(a)
 Вычисляет площадь квадрата.

 Параметры:> - a (float): Длина стороны квадрата.

 Возвращает:
 - float: Площадь квадрата.

Пример вызова:
 result = area(4)
 print(result)  # Вывод: 16.0

#### perimeter(a)
 Вычисляет периметр квадрата.

 Параметры:
 - a (float): Длина стороны квадрата.

 Возвращает:
 - float: Периметр квадрата.

 Пример вызова:
 result = perimeter(4)
 print(result)  # Вывод: 16.0
